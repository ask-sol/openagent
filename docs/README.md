# OpenAgent docs

The site at [ask-sol.github.io/openagent](https://ask-sol.github.io/openagent), built with Jekyll + the [Just the Docs](https://just-the-docs.com) theme. Hosted on GitHub Pages so the HTML is fully crawlable by Google Search Console (no JS-rendered routing).

## Local preview

```bash
cd docs
bundle install
bundle exec jekyll serve --livereload
# open http://localhost:4000/openagent/
```

## Adding a page

1. Create `docs/<section>/<page>.md`.
2. Add the front matter:
   ```yaml
   ---
   title: My page
   parent: Section name           # match the parent's title:
   nav_order: 5                   # ordering within parent
   ---
   ```
3. Push to `main`. The GitHub Action at `.github/workflows/docs.yml` rebuilds and deploys automatically.

## Custom domain

When you register a domain, drop a single line into `docs/CNAME` (just the apex hostname, no protocol) and configure DNS per [GitHub's docs](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site). Then update `url:` and `baseurl:` in `_config.yml` accordingly.

## Indexing on Google Search Console

1. Verify ownership at [search.google.com/search-console](https://search.google.com/search-console) using the HTML-tag method — paste the `<meta>` tag into `docs/_includes/head_custom.html` (create the file if it doesn't exist).
2. Submit `https://ask-sol.github.io/openagent/sitemap.xml` (auto-generated by `jekyll-sitemap`).
3. Request indexing of `/`, `/essentials/quickstart`, and `/providers` to seed.

## Theme customization

Just the Docs is configured in `_config.yml`. The most common knobs:

- `color_scheme: dark` — currently dark; change to `light` or define your own in `docs/_sass/`.
- `aux_links` — top-right links.
- `nav_external_links` — sidebar external links.
- `callouts` — typed admonition boxes (`> **Tip** ...{: .tip }`).
