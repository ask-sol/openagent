# Keybindings

| Key | Action |
|:---|:---|
| `↑` / `↓` | Cycle through your previous prompts |
| `Esc` | Stop current task / close menus / dismiss permission prompts |
| `Shift+Esc` | Queue a message while one is in flight |
| `Ctrl+T` | Toggle terminal mode (input box runs as shell) |
| `Ctrl+B` | Expand / collapse old messages |
| `Shift+Tab` | Cycle permission mode (Standard → Cautious → Unrestricted) |
| `Ctrl+C` | Cancel the current task; press again to quit |
| `Tab` | Autocomplete slash command |
| `Enter` | Submit |
